Backend examples

/examples/data:
Flask code fetching data from postgres without an ORM.

/examples/auth:
Integrates with Hasura's auth UI kit to add login/signup functionality without any code. Uses Hasura API gateway to checks if a user is logged in without any session-handling code.

/examples/filestore:
Allow users to upload files without writing file-handling code.
